# 🚀 AI Job Application Tracker

A React.js-based dashboard to manage and track job applications efficiently.

## Features

- Login Authentication
- Protected Routes
- Add, Edit, Delete Applications
- Search Applications
- Status Filtering
- Pagination
- Local Storage Persistence
- Dark Mode Toggle
- Dashboard Analytics
- Pie Chart Visualization
- Success Rate Tracking
- Responsive Design

## Tech Stack

- React.js
- JavaScript
- CSS3
- React Router DOM
- Recharts
- React Icons
- Local Storage
